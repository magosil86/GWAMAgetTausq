GWAMAgetTausquared

A tweaked version of GWAMA that reports the tau_squared heterogeneity statistic

INSTALLATION (MAC/UNIX)

Extract the tar.gz file:

tar xvzf gwama_reports_tau2.tar.gz

Change directory into the gwama folder:

cd gwama

Compile gwama:

make

Run gwama:

GWAMA
